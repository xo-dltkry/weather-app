# Weather App

A simple Weather App built with **React** that fetches real-time weather data from the OpenWeatherMap API (https://openweathermap.org/).  
The app allows users to search for any city and view current weather conditions such as temperature, humidity, wind speed, and weather description.

---

## Features
- Search weather by city name  
- Real-time data from **OpenWeatherMap API**  
- Displays:
  - Current temperature
  - Humidity  
  - Wind speed  
  - Weather description and icon  
- Responsive design  
- Error handling for invalid city names  

---

## Technologies Used
- **React** (Vite)
- **JavaScript (ES6+)**
- **CSS**
- **OpenWeatherMap API**
